# Aswamithra

Aswamithra is an agricultural marketplace platform connecting farmers, businesses, and customers.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Node.js + Express + MongoDB (backend)

## Local Development

1. Install dependencies:
```bash
npm install
```
2. Start frontend:
```bash
npm run dev
```
3. Build frontend:
```bash
npm run build
```
4. Run tests:
```bash
npm test -- --run
```

## Backend

From the `backend` directory:

1. Install dependencies:
```bash
npm install
```
2. Start server:
```bash
npm run dev
```

Configure environment variables using `.env.example` files for frontend and backend.
